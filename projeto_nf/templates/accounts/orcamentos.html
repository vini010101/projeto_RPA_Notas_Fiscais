Termos de Armazenamento de Dados

CONTRATO DE CONSENTIMENTO PARA ARMAZENAMENTO E TRATAMENTO DE DADOS PESSOAIS PARA ORÇAMENTO

1. OBJETO

1.1 A EMPRESA realizará um processo de orçamento para serviços solicitados pelo CLIENTE.

1.2 O CLIENTE concorda em fornecer à EMPRESA informações e dados pessoais necessários para a elaboração e consideração do orçamento.

2. AUTORIZAÇÃO DE USO DE DADOS

2.1 O CLIENTE autoriza a EMPRESA a coletar, armazenar e utilizar os dados fornecidos para fins de elaboração do orçamento e acompanhamento do serviço.

2.2 Os dados fornecidos pelo CLIENTE incluem, mas não se limitam a, nome, endereço, número de telefone, endereço de e-mail, descrição dos serviços desejados e outras informações relevantes para a elaboração do orçamento.

3. CONFIDENCIALIDADE E SEGURANÇA

3.1 A EMPRESA compromete-se a tratar todas as informações fornecidas pelo CLIENTE de forma confidencial e a adotar medidas adequadas para proteger esses dados contra acessos não autorizados, vazamentos ou qualquer forma de tratamento inadequado.

3.2 Os dados do CLIENTE serão acessíveis apenas aos responsáveis pelo orçamento e utilizados exclusivamente para fins relacionados à solicitação de orçamento.

4. PERÍODO DE RETENÇÃO DE DADOS

4.1 Os dados fornecidos pelo CLIENTE serão retidos pela EMPRESA durante o período necessário para a elaboração do orçamento e, se aplicável, durante a execução do serviço.

4.2 Caso o CLIENTE contrate os serviços, os dados fornecidos serão mantidos e geridos de acordo com as políticas internas da EMPRESA sobre gestão de dados pessoais.

5. DIREITOS DO CLIENTE

5.1 O CLIENTE tem o direito de solicitar acesso, correção ou exclusão dos seus dados pessoais a qualquer momento, entrando em contato com a EMPRESA através dos canais de comunicação disponibilizados.

6. CONSIDERAÇÕES FINAIS

6.1 Este contrato representa o entendimento completo entre as partes e substitui qualquer acordo ou entendimento anterior, oral ou escrito, relacionado ao objeto deste contrato.

6.2 Ao assinar este contrato, o CLIENTE reconhece que leu e compreendeu as condições aqui estabelecidas, concordando voluntariamente em fornecer seus dados pessoais para a elaboração do orçamento pela EMPRESA.
